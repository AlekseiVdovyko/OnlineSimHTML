ЗАДАЧА: Распарсить прайс лист со страницы http://onlinesim.ru/price-list, 
ВАЖНО: json-ответ не использовать, именно парсинг html-страницы.

Привести к виду: 
Map<Название страны, Map<Название сервиса, цена>>.

Привести в json строку. 
Полученные данные вывести в консоль или файл, или визуализировать (по  своему усмотрению) в окне написанном на Swing (JavaFx) 

Примерный результат: 
{ 
 "Россия" : { 
 "Вконтакте" : 2, 
 "Mail.ru" : 1 
 ... 
 }, 
 "Германия" : { 
 "Вконтакте" : 2, 
 "Mail.ru" : 1 
 ... 
 } 
 ... 
}


Решение:

1. В качестве браузера используется Chrome. Для смены браузера нужно заменить драйвер в папке selenium и путь к драйверу в строке 31.
Скачать драйверы: https://www.selenium.dev/documentation/webdriver/getting_started/install_drivers/
Версия драйвера должна соответствовать версии браузера.

2. Если прокручивание страницы сайта происходит слишком быстро или слишком медленно
(нажатие на кнопку может производиться только в видимой области окна браузера) нужно изменить скорость прокручивания на строке 72 deltaY:

3. Если очень плохая скорость Интернета, то в строке 37 увеличить время бездействия потока (по умолчанию 1000 Мс), чтобы страница успела
загрузиться до начала парсинга. Для уменьшения скорости нажатия раскомментировать строку 66.
